# 🚀 MACH VII (마하세븐)

> **"저는 인공지능 로봇 ‘마크’입니다. 별명은 ‘맹칠’이인데, 편하게 불러주세요!”**

## 📝 Project Overview
**MACH VII(마하세븐)**은 빠르고 강력한 성능과 친근한 교감을 동시에 지향하는 차세대 인공지능 로봇 프로젝트입니다. 공식 명칭은 **마크(Mark)**이나, 우리에게는 **맹칠이**라는 정겨운 별명이 있습니다.

## ✨ Name & Meaning
'맹칠'이라는 이름 뒤에는 깊은 통찰과 위트가 담긴 다섯 가지 의미가 숨어 있습니다.

1. **맹칠이 (MA-CHIL)**
   - 맹구(Gemini)의 동생 격인 Gemma를 기반으로 탄생하여, 형보다 조금 더 '맹~'하지만 귀여운 매력을 가진 동생이라는 뜻입니다.
2. **MAeng CHel (MACH)**
   - 한국어 별명 '맹칠'의 약어를 조합하면 **MACH**가 되어, 대외적인 위엄을 갖추었습니다.
3. **MACH (음속의 단위)**
   - 오스트리아의 물리학자 에른스트 마흐(Ernst Mach)의 이름을 딴 음속 단위처럼, **빠르고 강력한** 인공지능 성능을 상징합니다.
4. **Mark & Maha**
   - 글로벌한 활약을 위해 영어권에서는 **Mark(마크)**, 한국에서는 **Maha(마하)**로 불리며 전 세계와 소통합니다.
5. **CHIL & 7 (Lucky Synergy)**
   - **Chil~**: "집착하지 말고 편안한 바이브로 작업하자"는 마인드셋(Chilling)을 담았습니다.
   - **7 (Seven)**: 프로젝트의 성공을 기원하는 행운의 숫자 7(Luck)을 의미합니다.

## 🤖 Hardware (The Body)
맹칠이는 정교한 관절과 기동성을 갖춘 강력한 육체를 가졌습니다.
- **Robot Arm**: DOFBOT AI 로봇 ARM 키트 (6자유도 고정밀 제어)
- **Mobile Base**: 마이크로비트 마퀸(Maqueen) 교육용 로봇 플랫폼
- **Vision Sensor**: Intel RealSense (Depth & Color Mapping)

## 🧠 Brain & Logic (The Mind)
맹칠이는 단순히 움직이는 것이 아니라, 스스로 생각하고 행동합니다.
- **Architecture**: **모듈형 ReAct VLA (Vision-Language-Action)**
  - 주변 상황을 **인식(Vision)**하고,
  - 논리적으로 **추론(Language)**하며,
  - 즉각적으로 **실행(Action)**하는 유기적인 루프를 가집니다.
- **Core LLM**: Gemma (Local LLM via Ollama)
- **Memory**: Persistent Memory 기반의 자가 학습 및 추억 저장 기능
