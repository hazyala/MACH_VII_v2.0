### dofbot_simple.py

URDF 기반의 로봇 기구학 모델(IKPy)과 실제 DOFBOT 하드웨어 제어를 연결하는 핵심 제어 모듈이다.


dofbot_simple.py는 DOFBOT 로봇팔의 실제 서보모터 제어와 URDF 기반 기구학 계산을 하나의 인터페이스로 통합한 모듈이다.

내부적으로 Arm_Lib을 통해 실제 서보모터 각도를 읽고 제어하며, IKPy 라이브러리를 사용해 순기구학(FK)과 역기구학(IK)을 계산한다.

외부 코드에서는 관절 각도가 아닌 XYZ 좌표 단위로 로봇을 제어할 수 있도록 추상화되어 있으며, 하드웨어 상태와 기구학 모델을 항상 동기화하도록 설계되었다.

이 파일은 본 프로젝트의 로봇 제어 핵심 엔진 역할을 수행한다.



### robot_thread.py

로봇 제어를 메인 프로그램과 분리하여 안정적으로 실행하기 위한 전용 스레드 관리자.


### shared.py

스레드 간 로봇 상태와 명령을 안전하게 공유하기 위한 공용 데이터 공간.
